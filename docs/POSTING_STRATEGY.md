# X(Twitter) 投稿戦略ガイドライン
本ドキュメントはオープンソース化されたXのアルゴリズムコードと各種調査データをもとに作成。
このプロジェクトの自動投稿はこの原則に従って運用する。

---

## アルゴリズムの基本構造

Xのスコアリング式（オープンソースコードより）：

```
FinalScore = (Retweet × 20) + (Reply × 13.5) + (ProfileClick × 12)
           + (LinkClick × 11) + (Bookmark × 10) + (Like × 1)
           - ペナルティ
```

リツイート1件 = いいね20件分の価値。
返信の返信は最も高い重み（+75）を持ち、いいねの150倍の価値がある。

---

## TweepCredスコア（アカウント信頼スコア）

- 全アカウントに0〜100のスコアが付与される
- 閾値 0.65 未満：配信対象ツイートが3件のみに制限される
- 閾値 0.65 以上：全ツイートが配信対象になる
- Premiumアカウントは+4〜+16のブーストあり

スコアを上げる要因：
- アカウント年齢（30日以上で効果が出始める）
- フォロワー/フォロー比（フォロー数500以上かつ比率0.6超でペナルティ）
- エンゲージメント品質
- 高品質アカウントとのインタラクション

---

## ブースト要因（やるべきこと）

- 動画投稿（10秒以上視聴されると大きなブースト）
- 画像・GIF・ポール付き投稿
- 投稿後1時間以内に返信に反応する（最重要）
- ハッシュタグは1〜2個のみ（ニッチなもの）
- 数字・具体的なデータを含める
- スレッド形式で展開する

---

## ペナルティ要因（やってはいけないこと）

- 外部リンクを本文に直貼り → 30〜50%のリーチ減（無料アカウントは中央値エンゲージメントがゼロに近い）
  - 回避策：リンクは返信（リプ）に貼る
- ハッシュタグを3個以上使う → 40%ペナルティ
- 攻撃的なテキスト → 80%リーチ減
- 1日10投稿以上（低エンゲージメント投稿が多いとペナルティ）
- フォロー数500超かつフォロー/フォロワー比0.6超

## 時間帯戦略（JST基準）

### 株・投資アカウント向け最適時間帯

| 投稿内容 | 推奨時間（JST） | 理由 |
|---|---|---|
| 朝まとめ（米市場開場前） | 21:30〜22:30 | 米国市場9:00〜9:30 ET開場前、投資家が情報収集 |
| 夜総括（米市場終了後） | 05:30〜06:30 | 米国市場16:00 ET終了後の確定値 |
| 昼のエンゲージメント狙い | 12:00〜13:00 | 日本のランチタイム、最もスクロールされる時間帯 |
| 夕方の通勤時間 | 17:00〜19:00 | 帰宅時のスマホ閲覧ピーク |

### 曜日別優先度

- 火・水・木：最高優先（エンゲージメント最大）
- 月・金：中程度
- 土・日：最低（特に土曜は最も効果が低い）

## 投稿テンプレート

### 朝まとめ
```
【本日の注目銘柄】$TICKER

前日終値: $XX.XX（前日比 ±X.X%）
出来高: XXM（平均比 ±X%）

注目ポイント：
・（ツイートから収集した注目情報1）
・（ツイートから収集した注目情報2）

#米国株 #投資
```

### 夜総括
```
【本日の市場総括】

$TICKER: $XX.XX（±X.X%）
$TICKER2: $XX.XX（±X.X%）

本日のまとめ：
（100字以内の総括）

#米国株 #投資
```

---

## 運用ルール

- 1日の投稿上限：10件
- 投稿間隔：最低30分以上空ける
- リンクは本文に入れず返信に貼る
- 投稿後30分以内に来た返信には必ず反応する
- 重複投稿禁止（tmp/posted_ids.txt で管理）
- ハッシュタグは #米国株 #投資 の2個のみ
